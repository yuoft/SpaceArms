    变成基岩锭，基岩锭与空间核心合成基岩。基岩在玩家身上持续给予debuff，
    玩家合成基岩武器时有附魔，添加村民（武器匠）交易获取空间核心，添加末地

    亡灵生物附加瞬间治疗，非亡灵瞬间伤害，荧石攻击发光。上调龙晶剑和空间剑
    伤害。添加战利品表方块掉落，修改部分矿物为时运掉落。
